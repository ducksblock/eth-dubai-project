## How to Run
1. Install dependencies
```
npm i
yarn
```

2. Run the project in dev
```
npm run dev
yarn dev
```
3. Run the project in prod
```
npm run build
npm run start

yarn build
yarn start
```

The server will start at http://localhost:3000.
